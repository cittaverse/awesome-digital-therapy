## 为什么创建这个仓库？

**问题**：数字疗法领域信息高度分散，尤其是**老年认知健康**垂直领域缺乏系统资源整理。

**现有 awesome 列表的缺口**：
| 仓库                                                                               | 覆盖范围       | 缺口          |
|--------------------------------------------------------------------------------|----------|-----------|
| [awesome-mental-health](https://github.com/dreamingechoes/awesome-mental-health) | 软件行业心理健康   | ❌ 非数字疗法，非老年 |
| [awesome-ai-healthcare](https://github.com/awesomelistsio/awesome-ai-healthcare) | AI 医疗通用    | ❌ 不专注认知/老年  |
| [awesome-health](https://github.com/lalaithan/awesome-health)                    | 通用健康资源     | ❌ 无数字疗法     |

**我们的定位**：
- ✅ **唯一专注数字疗法 + 老年认知**的 awesome 列表
- ✅ **中文资源占位**（政策/产品/研究）
- ✅ **临床验证导向**筛选（非营销内容）

---

## 什么是数字疗法？

**数字疗法 (Digital Therapy/DTx)** 是基于软件程序的循证医学干预，可预防、管理或治疗疾病。

**本仓库聚焦**：
- 🧠 认知训练与 MCI 早期干预
- 👴 老年记忆衰退延缓
- 💬 语言/叙事评估工具
- 🤖 AI 辅助数字疗法

## 贡献指南

欢迎提交 PR 添加资源！

### 收录标准

1. **相关性**：必须与数字疗法/认知训练/老年护理直接相关
2. **质量**：优先同行评审论文、知名机构报告、活跃开源项目
3. **可访问性**：优先开放获取资源
4. **临床验证**：优先有临床数据支撑的产品/研究

### 添加格式

```markdown
### 分类名称

```

### 提交流程

1. Fork 本仓库
